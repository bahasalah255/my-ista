# 🎓 My-ISTA

**My-ISTA** is an educational web platform designed for ISTA students.  
It provides structured access to modules, quizzes, exams, and academic resources in a clean and organized interface.

---

## 🚀 Features

- 📚 Modules by subject
- 📝 Interactive quizzes
- 📊 Exams section
- 🎯 Clean navigation structure
- 📱 Responsive design
- 📄 Privacy & Terms pages
- 🗺️ Sitemap included

---

## 🛠️ Technologies Used

- HTML5
- CSS3 / Bootstrap
- JavaScript
- Responsive Web Design

---

## 🎯 Project Goal

The goal of this project is to create a centralized academic resource hub for ISTA students, helping them access learning materials easily.

---

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/bahasalah255/my-ista.git
   ```

2. **Navigate to the project folder**
   ```bash
   cd my-ista
   ```

3. **Open `index.html` in your browser**
   ```bash
   # Simply open the file directly or use a live server extension
   open index.html
   ```

> No build tools or dependencies required — it's a pure HTML/CSS/JS project!
