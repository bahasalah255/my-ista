# 🎓 My-ISTA

My-ISTA is an educational web platform designed for ISTA students.  
It provides structured access to modules, quizzes, exams, and academic resources in a clean and organized interface.

---

## 🚀 Features

- 📚 Modules by subject
- 📝 Interactive quizzes
- 📊 Exams section
- 🎯 Clean navigation structure
- 📱 Responsive design
- 📄 Privacy & Terms pages
- 🗺️ Sitemap included

---

## 🛠️ Technologies Used

- HTML5
- CSS3 / Boostrap
- JavaScript
- Responsive Web Design

## 🎯 Project Goal

The goal of this project is to create a centralized academic resource hub for ISTA students, helping them access learning materials easily.
