M106 - Manipuler des Bases de Données

Module Fondamental | Durée: 110h

Introduction

Comprendre les principes des bases de données et développer les compétences en manipulation de données.

Objectifs du module

  • Concevoir des schémas de base de données relationnels
  • Maîtriser SQL (SELECT, INSERT, UPDATE, DELETE)
  • Créer et gérer des tables et des relations
  • Utiliser les jointures et les agrégations

Contenu du module

  • Introduction aux bases de données
  • Le modèle relationnel et la normalisation
  • La conception logique et physique
  • SQL : DDL (CREATE, ALTER, DROP)
  • SQL : DML (SELECT, INSERT, UPDATE, DELETE)
  • Les jointures et les sous-requêtes
