M105 - Développer des Sites Web Dynamiques

Module Fondamental | Durée: 110h

Introduction

Créer des applications web dynamiques avec interaction serveur-client et gestion de bases de données.

Objectifs du module

  • Concevoir des sites web dynamiques
  • Utiliser un serveur web et un langage de script côté serveur
  • Gérer les sessions et les cookies
  • Interagir avec les bases de données
  • Implémenter les formulaires web

Contenu du module

  • Introduction à Php
  • Variables, types et opérateurs
  • Les structures de contrôle
  • Les fonctions et les closures
  • Le DOM et la manipulation d'éléments
  • Les événements et les écouteurs
  • La programmation asynchrone (Promises, async/await)
